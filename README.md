# BouwFlow MVP

Een eerste klikbaar frontend-prototype voor een geïntegreerde webapplicatie voor bouw- en infrastructuurbedrijven.

## Inbegrepen
- Directiedashboard
- CRM en relaties
- Opportuniteiten en aanbestedingen
- Calculatiedossiers
- Projectportfolio
- Responsieve mobiele weergave

## Starten
```bash
npm install
npm run dev
```

## Productiebouw
```bash
npm run build
npm run preview
```

Deze versie gebruikt demonstratiegegevens en bevat nog geen database, authenticatie of backend.
